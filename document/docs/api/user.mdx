# User API

## Login

### Login with email and password

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}> \users\login\password </b>
<br />
<br />

Return `access_token` and `refresh_token` from account, if tokens not exists create new one.

**REQUEST BODY**

| FIELD    | TYPE   | DESCRIPTION        |
| -------- | ------ | ------------------ |
| email    | string | Max 100 characters |
| password | string | Max 100 characters |

**EXAMPLE RESPONSE**

```json
{
	"data": {
		"access_token": "JWT Access token",
		"refresh_token": "JWT Refresh token"
	}
}
```

<br />

## Register

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}>\users\register</b>
<br />
<br />

Create a new account with provided `email` and `passwrd`. Return newly created `access_token` and `refresh_token`.

**REQUEST BODY**

| FIELD    | TYPE   | DESCRIPTION        |
| -------- | ------ | ------------------ |
| email    | string | Max 100 characters |
| password | string | Max 100 characters |

**EXAMPLE RESPONSE**

```json
{
	"data": {
		"access_token": "JWT Access token",
		"refresh_token": "JWT Refresh token"
	}
}
```

<br />

## Verify account

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}> \users\verify-otp </b>
<br />
<br />

Verify the account with OPT.

**REQUEST BODY**

| FIELD | TYPE   | DESCRIPTION     |
| ----- | ------ | --------------- |
| OTP   | string | 6 digits number |

<br />

## Resend verify account

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}> \users\resend-verify-otp</b>
<br />
<br />

Resend OTP to email.

**REQUEST HEADER**

| FIELD        | TYPE   | DESCRIPTION |
| ------------ | ------ | ----------- |
| access_token | string | JWT Token   |

<br />

## Forgot password

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}> \users\forgot-password </b>
<br />
<br />

Send a email contain OTP to `email`.

**REQUEST BODY**

| FIELD | TYPE   | DESCRIPTION        |
| ----- | ------ | ------------------ |
| email | string | Max 100 characters |

**EXAMPLE RESPONSE**

```json
{
	"data": {
		"forgot_password_token": "JWT Token"
	}
}
```

<br />

## Reset password

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}> \users\reset-password </b>
<br />
<br />

Change the password to new password when forgot password.

**REQUEST BODY**

| FIELD                 | TYPE   | DESCRIPTION                                           |
| --------------------- | ------ | ----------------------------------------------------- |
| forgot_passwork_token | string | JWT Token                                             |
| password              | string | Max 100 characters                                    |
| confirm_password      | string | Max 100 characters and must be the same as `password` |

<br />

## Change password

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}>\users\change-password</b>
<br />
<br />

Change the password to new password.

**REQUEST HEADER**

| FIELD        | TYPE   | DESCRIPTION |
| ------------ | ------ | ----------- |
| access_token | string | JWT Token   |

**REQUEST BODY**

| FIELD            | TYPE   | DESCRIPTION                                           |
| ---------------- | ------ | ----------------------------------------------------- |
| old_password     | string | Max 100 characters                                    |
| password         | string | Max 100 characters                                    |
| confirm_password | string | Max 100 characters and must be the same as `password` |

# Refresh token

<span style={{ backgroundColor: '#49CC90', borderRadius: '10px', padding: '1px 6px' }}>POST</span>
<b style={{ fontSize: '1.5rem' }}>\users\refresh_token</b>
<br />
<br />

Get a new `access_token` and `refresh_token`.

**REQUEST BODY**

| FIELD         | TYPE   | DESCRIPTION |
| ------------- | ------ | ----------- |
| refresh_token | string | JWT Token   |

**EXAMPLE RESPONSE**

```json
{
	"data": {
		"access_token": "JWT Access token",
		"refresh_token": "JWT Refresh token"
	}
}
```
